# Evidence Index

此目录只保留可复查、脱敏的命令输出和截图说明。

- `local-uat/`：由 `npm run uat` 生成的本地 API UAT 输出。它使用 `AI_PROVIDER=demo`，只能证明应用链路，不是真实模型调用证据。
- `pair-uat-2026-07-22/`：浏览器双会话自动化 UAT，22/22 通过；用于证明界面、持久化与错误路径，不替代真人同伴反馈。
- `real-ai-2026-07-22/`：脱敏的真实模型生成、不同模型审查和 Markdown 导出摘要。
- `human-uat-2026-07-23/`：项目本人人工验收截图及同名说明；包含修复前故障复现和修复后通过截图，不能混用。
- 真实模型验证：在获得授权和 `.env.local` 配置后，保存脱敏状态码、模型名、时间、输入摘要、输出结构摘要和失败情况；不得保存 API Key、Authorization header、完整敏感想法或支付信息。
- 独立审查：应另外保存审查人/模型身份、只读提示词摘要、结论和后续修正的可复查记录。

人工保存或独立提交的截图须配同名 Markdown 说明文件，写明界面状态、脱敏处理和可替代验证命令。自动连续 UAT 截图可由同目录的结构化结果文件（例如 `uat_results.json`）统一索引，但必须记录步骤、结果与截图路径。
