在仓库根目录执行：

```bash

npm install

```

浏览器打开：<http://127.0.0.1:3000>

- 使用 `demo` provider 即可完成本次真人体验 UAT；它只验证界面和业务流程。
- 真实模型生成/审查应单独引用 `evidence/real-ai-2026-07-22/`，不要在本记录写入任何 token、私有 endpoint 或敏感想法。
